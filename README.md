```bash

```

Optional: preview the production build locally:

```bash
npm run preview
```

## Data & privacy

**中文：** 数据仅保存在本机浏览器的 IndexedDB 中；请使用同一浏览器配置文件访问；不提供云端同步。

**English:** Data is stored only in the browser’s IndexedDB. Use the same browser profile to see your entries. There is no cloud sync.
